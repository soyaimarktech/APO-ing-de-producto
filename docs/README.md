# docs/

Documentación adicional de APO que no forma parte de los documentos raíz.

Sugerido conforme crezca el producto:
- `diagrams/` — diagramas de arquitectura y flujo.
- `roadmap/` — planeación por versión.
- `benchmarks/` — mediciones de rendimiento.

> Las decisiones de arquitectura (ADR) viven por ahora en `DECISIONS.md` (raíz). Si crecen, se pueden
> mover a `docs/adr/`.
